# 并行-Parallel

将多个动画放到数组里面，并行运行，数值也是按照数组形式输出

# 引入

import { parallel } from 'popmotion';

# 使用

parallel(
  tween({ from: 40, to: 50 }),
  spring({ to: 500 })
).start(([ tweenOutput, springOutput ]) => {});

tweenOutput就是tween输出的值

springOutput就是spring输出的值
